# 🎓 Student Performance Intelligence System

An AI-powered academic analytics dashboard built using Streamlit and Machine Learning to analyze student performance, identify at-risk students, and generate intelligent insights for educators.

---

## 🚀 Live Demo

👉 Open Deployed App  
https://student-performance-intelligence-system.streamlit.app/

---

## 📌 Features

- 📊 Interactive Student Analytics Dashboard  
- 🚨 Early Warning & Risk Detection System  
- 🏆 Student Leaderboard & Ranking  
- 📈 Subject-wise Performance Analysis  
- 📉 Pass/Fail Prediction  
- 🧠 Machine Learning Based Insights  
- 📄 PDF Report Generation  
- 👩‍🏫 Teacher Dashboard  
- 📂 CSV Dataset Upload Support  
- 📱 Responsive Streamlit Interface  

---

## ▶️ Run Locally
1. Clone the repository
git clone https://github.com/your-username/student-performance-intelligence-system.git
2. Navigate to the project folder
cd student-performance-intelligence-system
3. Install dependencies
pip install -r streamlit-app/requirements.txt
4. Run the Streamlit app
streamlit run streamlit-app/app.py
