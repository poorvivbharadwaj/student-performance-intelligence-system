# 🎓 Student Performance Intelligence System

An AI-powered dashboard for analyzing student academic performance using Machine Learning, Streamlit, and Data Analytics.

## 🚀 Features

- 📊 Student Performance Dashboard
- 🤖 AI Chatbot
- 📈 Trend Analysis
- 🚨 Early Warning System
- 🏆 Leaderboard
- 👩‍🏫 Teacher Dashboard
- 📄 PDF Report Generation
- 🔍 Multi-Student Comparison
- 📉 Correlation Analysis
- 🌐 Gemini AI Integration

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Plotly
- ReportLab
- Gemini AI

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
